Kiến trúc dự án CloudFormation là gì? CloudFormation là một dịch vụ Infrastructure as Code được phát triển bởi Amazon Web Service, nó cho phép chúng ta define all resouces và tự động triển khai trên AWS bằng cách thông qua giao diện dòng lệnh. Việc sử dụng CloudFormation mang lại nhiều lợi ích ví dụ như: Triển khai được trên nhiều môi trường. Tối ưu hóa thời gian. Nhất quán và dễ dàng triển khai. Tái sử dụng templates. CloudFormation sử dụng 2 loại type: yaml or json ( trong dự án này mình sử dụng loại type yaml vì nó dễ nhìn). ...
